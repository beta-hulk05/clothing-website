## 🛒 Featured Project: RNY Clothing E-commerce Platform

<div align="center">
  <img src="https://github.com/beta-hulk05/project-screenshots/raw/main/rny-banner.png" alt="RNY Clothing Banner" width="800"/>
</div>

RNY Clothing is a full-featured e-commerce platform designed for a clothing brand that offers premium oversized tees and streetwear. The project consists of a customer-facing frontend and an admin dashboard for managing products, orders, and inventory.

### ✨ Key Features

#### Customer Portal
- **Responsive Design**: Fully optimized shopping experience across all devices
- **User Authentication**: Secure login/registration system with profile management
- **Product Catalog**: Browse products with filtering by category, size, and price
- **Product Details**: High-quality image gallery, size selection, detailed descriptions
- **Shopping Cart**: Real-time cart management with persistent storage
- **Customer Reviews**: Leave ratings and reviews for purchased products
- **Checkout Process**: Streamlined ordering with address form and payment options
- **Order Tracking**: View and track order status from placement to delivery
- **Easy Returns**: 7-day hassle-free exchange policy

#### Admin Dashboard
- **Product Management**: Add, edit, delete, and manage product listings
- **Inventory Control**: Manage stock levels and product availability
- **Order Processing**: View orders and update order status
- **User Management**: Manage customer accounts and access permissions
- **Analytics Dashboard**: Track sales, popular products, and customer behavior
